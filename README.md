# 📖 English-Dictionary

A simple application where user can search **English Words meaning**.

## 🚀 Features

- ✅ Search bar
- ✅ Displays:
  - Noun of the word
  - Meaning
  - Synonyms
- ✅ Simple design using HTML, CSS, and JavaScript

---

## 🛠️ Tech Stack

- **HTML** — Page structure
- **CSS** — Styling and layout
- **JavaScript** — Fetch data from API and dynamic DOM updates
